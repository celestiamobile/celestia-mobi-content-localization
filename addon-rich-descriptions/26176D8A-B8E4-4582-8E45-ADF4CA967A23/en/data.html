This addon adds a high-resolution virtual texture (VT) for the Galilean moon Io, which replaces the default texture. Includes surface map, normal map and night map, created from imagery from the Voyager and Galileo missions. Additionally, includes XYZV trajectories that ensure more precision for Galileo and Juno flybys.

Important

This addon includes VT textures and XYZV trajectories for Io. If you only want the trajectories and don't want VT textures, install only the Galilean Satellites (1995-2036) addon. Choose only one option to avoid possible conflicts.
