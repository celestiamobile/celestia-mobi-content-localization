This add-on presents 16 alternate surfaces of Earth, providing spectacular views of the planet as it might have looked like between 560 and 2,500 million years ago

Important

  • It is highly recommended that you also install RVS's Earth addon, so that the default Earth texture matches these alternate surfaces.
  • To use this add-on, make a long-press on Earth and find the tab "Alternate Surfaces", then a pop-up menu will appear letting you select which time period you'd like to see
