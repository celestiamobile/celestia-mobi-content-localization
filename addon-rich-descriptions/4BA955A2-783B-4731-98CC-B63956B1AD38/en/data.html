This addon adds a high-resolution virtual texture (VT) for the Galilean moon Europa, which replaces the default texture. Includes a surface map created from imagery from the Voyager, Galileo and Juno missions. Additionally, includes XYZV trajectories that ensure more precision for Galileo, Juno, Europa Clipper and JUICE flybys.

Important

This addon includes VT textures and XYZV trajectories for Europa. If you only want the trajectories and don't want VT textures, install only the Galilean Satellites (1995-2036) addon. Choose only one option to avoid possible conflicts.
