This addon adds a high-resolution virtual texture (VT) for the planet Venus, which replaces the default texture. Includes surface map and a normal map, created from radar data from the Magellan orbiter. Additionally, it also includes a new cloud texture.

Important

  • To see Venus's surface, remember to turn off atmospheres and clouds in Settings/Features.
  • Venus was mapped in detail only by the Magellan mission in the 1990s using synthetic-aperture radar. However, planetary coverage is not entirely perfect. Note that there will be some inconsistencies, holes and lack of data in some regions of the planet.
