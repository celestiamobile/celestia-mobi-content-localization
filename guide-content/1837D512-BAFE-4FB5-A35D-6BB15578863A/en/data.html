We have just released Celestia for Horizon OS (Meta Quest 2/3/3S/Pro). The app is available for free. Based largely on the Android version of Celestia, it is built with OpenXR to support the fully immersive experience. If you have one of these devices, you can get it from this link or by searching for "Celestia" in Meta Quest Store.

This is the first release, and it is expected to be buggy. Please report any issues you encounter to celestia.mobile@outlook.com.
