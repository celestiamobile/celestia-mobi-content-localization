Here you can find answers to the most asked questions for Celestia. Celestia also has a Wikibook page which you can turn to for additional resources.

How do I expand the content in Celestia?

Celestia has support for add-ons. Add-ons are content created by the community to enrich the content in Celestia. You can find out about where to find and how to use add-ons in Use Add-ons and Scripts.

I want to create my own add-ons, how do I get started?

Check out Addon Tutorials for Beginners on Wikibooks for making add-ons from scratch. Please also read Use Add-ons and Scripts to find out where to put your newly created add-ons.

I want to download all available add-ons from celestia.mobi, how do I achieve that?

Celestia does not offer a way to download multiple add-ons and we don't think you should try to do that either.
Some add-ons can have conflicts with other add-ons and can prevent Celestia from starting up or behaving correctly. Having too many add-ons also will slow down Celestia significantly.

There are errors in Celestia data or with the app itself, how do I report these errors?

If you notice errors with an add-on, you can make an issue in celestiamobile/celestia-addon-validator.
If it is an error with the data that comes with Celestia, not from add-ons, please post an issue in CelestiaProject/CelestiaContent.
If you want to report an issue with Celestia program, you can report the issue from inside the app in "Send Feedback" → "Report a Bug" or "Help" → "Report a Bug".

I can't find a certain astronomical object in Celestia. Why is that?

Celestia app's data is not up to date and that's usually why you can't find an object that appears on news. The community might have created add-ons for the object you are looking for, so you can browse the online add-ons to see if that is the case. Additionally Celestia's search feature only searches stars, DSOs, and objects in the nearest system, you might have to travel to the system of the object first.

I'm on Android 11 or later, how do I access the add-on folder?

By default, add-ons are installed to Android/data and Android 11 makes it hard for file managers to access the content under Android/data. However, Celestia (1.7.8 or later) has support for using Android/media folder as the default folder for add-ons.
To change the default folder used for add-ons, please go to "Settings" → "Data Location" and choose "Migrate Add-on Data". Be sure to create a backup before migrating your data to avoid data loss.
