安装 Celestia

在这里，你可以获取到最新版本的 Celestia。

如果你打开的是 celestia.mobi 提供的插件，而且你使用的 Celestia 版本支持直接打开插件，你可以点击页面中的“在 Celestia 中打开”。

如果你从别的地方下载了插件，请继续阅读。

使用默认的插件文件夹（推荐）

Celestia 在加载过程中会在一个特定的文件夹读取插件和脚本。在一些平台上这个文件夹的名称为“CelestiaResources”，这个文件夹下面包含“extras”文件夹（插件）和“scripts”文件夹（脚本）。

iOS/iPadOS

文件夹位于 App 的共享文件夹中，通过以下方式访问。

  1. 运行 Celestia 至少一次
  2. 打开“文件” App
  3. 切换到“浏览”选项卡
  4. 点击“我的 iPhone”或者“我的 iPad”
  5. 找到 Celestia 并打开

其中应该包含了 CelestiaResources 文件夹。

Android

文件夹位于 App 的外部文件夹，即 Android/data/space.celestia.mobilecelestia/files/CelestiaResources

在 Android 11 以上系统，此文件夹可能不可见，这种情况下可以在 Android/media/space.celestia.mobilecelestia 下面自行创建 CelestiaResources/extras 和 CelestiaResources/scripts 文件夹。

Mac (Catalyst)

文件夹位于 /Users/YOUR USER NAME HERE/Library/Containers/space.celestia.MobileCelestia/Data/Documents/CelestiaResources

把 "YOUR USER NAME HERE" 替换成你的用户名。

Mac (Legacy)

文件夹位于 /Users/YOUR USER NAME HERE/Library/Containers/space.celestia.Celestia/Data/Documents/CelestiaResources

把 "YOUR USER NAME HERE" 替换成你的用户名。

Windows (微软商店)

  1. 运行 Celestia
  2. 文件 -> 打开自定义文件夹

CelestiaResources 文件夹应该位于打开的文件夹中。路径应该类似 C:\Users\YOUR USER NAME HERE\AppData\Local\Packages\40507LinfengLi.Celestia_r7f2qt5a05s84\LocalState\CelestiaResources（把 "YOUR USER NAME HERE" 替换成你的用户名）。

Windows (Legacy)

在 Celestia 安装位置（默认为 C:\Program Files (x86)\Celestia 或者 C:\Program Files\Celestia）应该有“extras”和“scripts”文件夹

提供自定义数据文件夹

要实现完全的数据自定义，请查看提供自定义数据文件夹。
