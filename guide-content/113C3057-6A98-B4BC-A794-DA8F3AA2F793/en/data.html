New Celestia

Android

The best place to get Celestia for Android is on Google Play.

You can join beta testing here. Celestia for Android is also available as an APK file you can download on GitHub.

If you don't have Google Play, you can still get Celestia from alternative markets. Please note that there might be some differences in features in Celestia distributed in different markets.

iOS/macOS

Celestia for iOS/macOS is available on App Store.

You can join beta testing via TestFlight. You can also download Celestia for macOS directly on GitHub.

Meta Quest

Celestia for Meta Quest is available on Meta Quest Store for Meta Quest 2/3/3S/Pro. You can join beta testing here.

visionOS

Celestia for visionOS is provided exclusively on App Store.

You can join beta testing via TestFlight.

Windows/Xbox

Celestia for Windows/Xbox is available on Microsoft Store.

You can also sideload with the version provided on GitHub.

Web/Browser

Not ready to install Celestia on your device yet? You can try a lite version of Celestia in your browser.

Celestia (1.7.0)

All flavors of Celestia above are developed based on the original Celestia project version 1.7.0. You can get a nightly Windows installer of original Celestia 1.7.0 from GitHub.

Celestia (1.6.x)

The latest stable release of the original Celestia project (1.6.x series) is 1.6.4, released in November 2023. It is available on Linux (source code), macOS, and Windows.
