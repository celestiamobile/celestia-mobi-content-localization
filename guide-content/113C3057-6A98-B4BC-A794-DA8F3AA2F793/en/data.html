New Celestia

Android

The best place to get Celestia for Android is on Google Play.

You can join beta testing here. Celestia for Android is also available on Amazon Appstore and as a direct download from GitHub.

If you don't have Google Play, you can still get Celestia from alternative markets. Please note that there might be some differences in features in Celestia distributed in different markets.

iOS

Celestia for iOS is provided exclusively on App Store.

You can join beta testing via TestFlight.

macOS

Celestia for macOS is available on Mac App Store.

You can join beta testing via TestFlight. You can also download Celestia directly on GitHub.

visionOS

Celestia for visionOS is provided exclusively on App Store.

You can join beta testing via TestFlight.

Windows/Xbox

Celestia for Windows/Xbox is available on Microsoft Store.

You can also sideload with the version provided on GitHub.

Celestia (1.6.x)

The latest version of Celestia (1.6.x series) is 1.6.4, released in November 2023. It is available on Linux (source code), macOS, and Windows
