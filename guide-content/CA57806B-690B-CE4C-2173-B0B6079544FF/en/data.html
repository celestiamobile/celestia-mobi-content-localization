Celestia supports full customization too. You can override the directory for Celestia to read from.

For all platforms, we need to first download a copy of the data directory of the Celestia release.

  1. Download the original copy of the data directory with the app version here
  2. Make modification to the directory, and celestia.cfg file

Then...

Android/iOS/macOS(Catalyst)

  1. Copy the directory to the device, for iOS it can be in an arbitrary directory, for Android it has to be under Android/data/space.celestia.mobilecelestia/files or Android/media/space.celestia.mobilecelestia
  2. Run Celestia, go to Settings -> Data Location
  3. Choose Data Directory location (CelestiaResources directory) and Config File location (celestia.cfg)
  4. Close Celestia and rerun

macOS(Legacy)

  1. Copy the directory to any place
  2. Run Celestia, go to MenuBar -> Celestia -> Change Config File
  3. Choose Data Directory location (CelestiaResources directory) and Config File location (celestia.cfg)
  4. Click "Confirm and Relaunch", note that the relaunching needs to performed by yourself

Windows(UWP)

  1. Rename CelestiaResources to Override
  2. Run Celestia and click File -> Open Custom Folder
  3. File explorer should be opened, copy the Override directory into it
  4. Close Celestia and rerun

If the configuration and data files are correctly set up, you should be able to see Celestia loading your files.
