Quick start: a .celx script runs as a Lua coroutine. Time-blocking is done with wait(seconds); the script can call any of the methods below to manipulate the scene. The top-level objects available at script start are celestia (the simulation), and accessors that fetch others like celestia:getobserver() for the camera.
